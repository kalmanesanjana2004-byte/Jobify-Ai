# 🎯 Jobify AI — Intelligent Job Recommendation System

Jobify AI is a smart job recommendation platform that uses machine learning to match users with the most relevant job opportunities based on their skills, experience, and qualifications. It helps users discover suitable career paths and identify skill gaps for improvement.

---

## 🚀 Features

- 📄 **Resume Upload & Parsing** (PDF supported)
- ✍️ **Manual Skill Entry**
- 🧠 **TF-IDF Based Text Analysis**
- 📊 **Cosine Similarity Matching**
- 🎯 **Job Match Score Calculation**
- 🔍 **Skill Gap Analysis**
- ⚡ **Fast & Interactive UI**
- 📱 **Responsive Design**

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Machine Learning:** scikit-learn (TF-IDF, Cosine Similarity)  
- **Other:** Jinja2 Templates
